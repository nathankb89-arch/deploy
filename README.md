💰 Finance+ Dashboard

A modern, multi-page Finance Web Application UI built using HTML and CSS.
This project showcases a clean and interactive fintech dashboard design with multiple pages and a consistent user experience.

📌 Features

- 🏠 Welcome Page (Landing Page)
  
  - Full-screen hero section with background image
  - Call-to-action button
  - Smooth animations

- 📊 Dashboard
  
  - Financial summary cards (Balance, Income, Expenses)
  - Clean layout and modern UI

- 💳 Wallet Page
  
  - Card-style financial accounts
  - Savings and investment sections

- 💸 Transactions Page
  
  - Transaction history list
  - Income and expense indicators

- 📈 Analytics Page
  
  - Visual progress bars (CSS-based)
  - Financial insights and summaries

- 👤 Profile Page
  
  - User information display
  - Account and security sections
